### 🔧 What I've worked on

- 🚀 **[Logfire](https://github.com/Yokiatch/logfire)** — A C++20 high-performance log analytics engine using `mmap` and single-pass filtering. ~476 MB/s throughput, ~93ms median query latency (44MB benchmark). Integrates RE2 for regex matching, exposes core to Python via `pybind11`, includes benchmark dashboards vs. Python/grep/ripgrep.
- 📦 **[CRoaring](https://github.com/RoaringBitmap/CRoaring)** — Added `mask()` and `maskClosed()` interval methods to the Roaring Bitmap C library ([PR #816](https://github.com/RoaringBitmap/CRoaring/pull/816), 67/67 tests passing, merged ✅)
- 🧮 **[NForge](https://github.com/Marbjo07/NForge/pull/51)** — Implemented matrix multiplication pipeline with CPU backend, batch broadcasting, CUDA stub, and Catch2 test suite
